# 💰 Smart Finance Tracker

A simple yet powerful Excel/Google Sheets based personal finance tracker to manage:

* 📅 Daily & Monthly Expenses
* 💰 Salary & Income
* 📈 Investments (SIP, Stocks, Mutual Funds, etc.)
* 🏦 Savings & Net Worth
* 📊 Monthly & Yearly Financial Analysis
* ⚠️ Smart Spending Alerts
* 📉 Interactive Dashboard & Charts

---

# 🚀 Features

* Track expenses by category:

  * UPI Expenditure
  * Food Purchase
  * Medicine Purchase
  * Ecommerce Purchase
  * Kitchen/Eatery Purchase
  * Rent, EMI, Electricity, etc.

* Separate tracking for:

  * Expenses
  * Investments
  * Savings

* Automatic:

  * Monthly summaries
  * Yearly analysis
  * Savings calculation
  * Net worth tracking
  * Overspending alerts

* Dynamic category support:
  Easily add new expense, income, or investment categories anytime.

---

# 📂 Sheets Included

| Sheet                | Purpose                       |
| -------------------- | ----------------------------- |
| `Transactions`       | Main data entry               |
| `Dashboard`          | Visual analytics & charts     |
| `Monthly Summary`    | Month-wise financial tracking |
| `Monthwise Analysis` | Detailed expense breakdown    |
| `Yearly Analysis`    | Year-wise financial overview  |

---

# 🧠 Financial Logic

```text id="q6f4c6"
Income - Expenses - Investments = Savings
```

```text id="8cl3s7"
Net Worth = Savings + Investments
```

---

# ✍️ How To Use

1. Open `Transactions`
2. Add:

   * Date
   * Category
   * Description
   * Amount
   * Type (`Income`, `Expense`, `Investment`)
3. Dashboard & reports update automatically.
